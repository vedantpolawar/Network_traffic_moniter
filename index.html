Space Complexity: Why O(B) is Bounded
RING BUFFER: O(B)
Fixed array of size B.
Never grows beyond B.
IPv4: B × 4 bytes
IPv6: B × 16 bytes
HASH TABLE: O(U) where U ≤ B
Only IPs in the ring
buffer have count > 0.
U ≤ B → bounded by B.
~28 bytes per entry.
MIN-HEAP: O(K)
Fixed K slots.
K << U typically.
~32 bytes per node.
Total: K × 32 bytes.
TOTAL SPACE = O(B + K) · Choosing B = 10,000 → <400 KB regardless of stream length n
